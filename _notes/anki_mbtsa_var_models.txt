What is a multivariate time series and what new phenomenon does it enable modeling?	A <b>multivariate time series</b> is a vector \(X_t = (X_{1t}, \ldots, X_{Nt})^\top \in \mathbb{R}^N\) representing \(N\) simultaneously recorded variables over time.<br><br>The primary advantage over univariate analysis is the ability to model <b>interactions between time series</b>, in particular the <b>cross-correlation</b> between component series.	mbtsa::var::introduction
Define the VAR(\(p\)) model and its components.	The <b>VAR(\(p\))</b> model:<br><br>\(X_t = c + \sum_{i=1}^p A_i X_{t-i} + \varepsilon_t\)<br><br><b>Intercept:</b> \(c \in \mathbb{R}^N\).<br><b>Coefficient matrices:</b> \(A_i \in \mathbb{R}^{N \times N}\) for each lag \(i = 1, \ldots, p\).<br><b>Error term:</b> \(\varepsilon_t \sim \mathcal{N}(0, \Sigma_\varepsilon)\), multivariate white noise.<br><br>Each variable is expressed as a linear function of its own past values, the past values of <b>all other variables</b>, and a random error.	mbtsa::var::definition
What is the error covariance matrix \(\Sigma_\varepsilon\) in a VAR model, and what do its off-diagonal elements mean?	\(\Sigma_\varepsilon = \mathbb{E}[\varepsilon_t \varepsilon_t^\top]\), with entries \((\Sigma_\varepsilon)_{jk} = \mathbb{E}[\varepsilon_{jt}\varepsilon_{kt}]\).<br><br>The <b>off-diagonal elements</b> are allowed to be non-zero, meaning the contemporaneous error terms for different variables can be <b>correlated</b> — shocks to different variables at the same time step may be related.<br><br><b>Empirical estimate:</b> \(\hat{\Sigma}_\varepsilon = \frac{1}{T-(p+1)} \sum_{t=p+2}^T \epsilon_t \epsilon_t^\top\)	mbtsa::var::covariance
What do the diagonal and off-diagonal entries of a VAR coefficient matrix \(A_i\) represent?	\(A_i = (a_{jk}^{(i)})_{j,k=1}^N\)<br><br><b>Diagonal entries</b> \(a_{jj}^{(i)}\): relate variable \(j\) to <b>its own past</b> at lag \(i\) — capture internal time constants and autoregressive properties of each series.<br><br><b>Off-diagonal entries</b> \(a_{jk}^{(i)}\) (\(j \neq k\)): quantify how the past of variable \(k\) influences the present of variable \(j\) — capture <b>cross-series dynamics</b> and interactions. This is the basis for <b>Granger causality</b>.	mbtsa::var::coefficient_matrices
State the two companion form equivalences for VAR models.	(1) Any <b>scalar AR(\(p\))</b> process can be written as a <b>\(p\)-variate VAR(1)</b> process.<br>(2) Any <b>VAR(\(p\)) in \(K\) variables</b> can be written as a <b>\(Kp\)-variate VAR(1)</b> process.<br><br><b>Significance:</b> This allows us to study stability and properties of any high-order VAR model by analyzing a <b>single, larger coefficient matrix</b> of the equivalent VAR(1) representation.	mbtsa::var::companion_form
State and sketch the proof of the stationarity condition for a VAR(1) process.	For \(X_t = c + AX_{t-1} + \varepsilon_t\), the stationarity condition is:<br><br>\(\max_i |\lambda_i(A)| < 1\)<br><br><b>Proof sketch:</b><br>(1) Iterate: \(X_t = A^t X_0\) (ignoring \(c, \varepsilon\)).<br>(2) Eigendecompose: \(A = V\Lambda V^{-1}\), so \(A^t = V\Lambda^t V^{-1}\).<br>(3) \(X_t = V\Lambda^t V^{-1} X_0\).<br>(4) If \(|\lambda_i| < 1\) for all \(i\), then \(\lambda_i^t \to 0\), so \(A^t \to 0\) and the process is stable.<br>If \(|\lambda_i| > 1\) for some \(i\), that mode grows exponentially — non-stationary.<br>If \(|\lambda_i| = 1\): marginally stable (e.g., random walk).	mbtsa::var::stationarity
How are parameters estimated for a VAR(\(p\)) model?	Parameters \((c, A_1, \ldots, A_p, \Sigma_\varepsilon)\) are estimated via <b>Maximum Likelihood Estimation</b>.<br><br>Log-likelihood: \(\ell(\theta) = \sum_{t=p+1}^T \log\, p(X_t \mid X_{t-1}, \ldots, X_{t-p}, \theta)\)<br><br>For <b>Gaussian errors</b>, each conditional is:<br>\(p(X_t \mid \cdot) = \mathcal{N}\!\left(c + \sum_{i=1}^p A_i X_{t-i},\; \Sigma_\varepsilon\right)\)<br><br>In this case, MLE is equivalent to <b>multivariate least squares</b> — the problem can be framed as a multivariate linear regression.	mbtsa::var::estimation
