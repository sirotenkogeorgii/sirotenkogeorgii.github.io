What are the four key components of a complete statistical model?	(1) <b>Model Architecture:</b> the mathematical form of the relationship between variables.<br>(2) <b>Loss Function:</b> quantifies error between predictions and actual data.<br>(3) <b>Training Algorithm:</b> optimization procedure to find parameters minimizing the loss.<br>(4) <b>Data:</b> observations used to train and evaluate the model.	mbtsa::regression::model_components
Write the univariate linear regression model and define each term.	\(y_t = x_t^\top \beta + \epsilon_t\)<br><br>\(y_t \in \mathbb{R}\): response variable at time \(t\).<br>\(x_t \in \mathbb{R}^{p+1}\): predictor vector (including intercept term).<br>\(\beta \in \mathbb{R}^{p+1}\): parameter/coefficient vector.<br>\(\epsilon_t \sim \mathcal{N}(0, \sigma^2)\): i.i.d. Gaussian error term.<br><br>This implies \(y_t \mid x_t, \beta, \sigma^2 \sim \mathcal{N}(x_t^\top \beta, \sigma^2)\).	mbtsa::regression::linear_regression
Write the vectorized form of the linear regression model.	\(Y = X\beta + E\)<br><br>\(Y = (y_1, \ldots, y_T)^\top\): \(T \times 1\) response vector.<br>\(X\): \(T \times (p+1)\) design matrix (rows are \(x_t^\top\)).<br>\(\beta = (\beta_0, \ldots, \beta_p)^\top\): \((p+1) \times 1\) parameter vector.<br>\(E = (\epsilon_1, \ldots, \epsilon_T)^\top\): \(T \times 1\) error vector.	mbtsa::regression::linear_regression
Write the Least Squares Error (LSE) loss function in both scalar and vector notation.	<b>Scalar:</b> \(\mathrm{LSE}(\beta) = \sum_{t=1}^T (y_t - x_t^\top \beta)^2\)<br><br><b>Vector:</b> \(\mathrm{LSE}(\beta) = (y - X\beta)^\top(y - X\beta)\)<br><br>The objective is to find \(\beta\) that minimizes this quantity.	mbtsa::regression::loss_function
State and derive the OLS estimator.	<b>OLS estimator:</b> \(\hat{\beta} = (X^\top X)^{-1} X^\top y\)<br><br><b>Derivation:</b><br>Expand: \(\mathrm{LSE}(\beta) = y^\top y - 2\beta^\top X^\top y + \beta^\top X^\top X \beta\)<br>Differentiate: \(\frac{\partial \mathrm{LSE}}{\partial \beta} = -2X^\top y + 2X^\top X \beta\)<br>Set to zero: \(X^\top X \hat{\beta} = X^\top y\)<br>Solve: \(\hat{\beta} = (X^\top X)^{-1} X^\top y\)	mbtsa::regression::ols
Why do the MLE and LSE solutions coincide for linear regression?	Under i.i.d. Gaussian errors, the log-likelihood is:<br>\(\ell(\beta) = \sum_t \left[-\frac{1}{2}\log(2\pi\sigma^2) - \frac{(y_t - x_t^\top\beta)^2}{2\sigma^2}\right]\)<br><br>Terms involving \(\sigma^2\) and \(2\pi\) are constant w.r.t. \(\beta\), so:<br>\(\arg\max_\beta \ell(\beta) \equiv \arg\min_\beta \sum_t (y_t - x_t^\top\beta)^2\)<br><br>Therefore \(\hat{\beta}_{\mathrm{LS}} = \hat{\beta}_{\mathrm{MLE}}\).	mbtsa::regression::mle
What are the core assumptions of the linear regression model on the errors?	(1) <b>Linearity:</b> the relationship between predictors and response is linear.<br>(2) <b>Normality of errors:</b> \(\epsilon_t \sim \mathcal{N}(0, \sigma^2)\).<br>(3) <b>Constant variance (homoscedasticity):</b> \(\mathrm{Var}(\epsilon_t) = \sigma^2\) for all \(t\).<br>(4) <b>Independence of errors:</b> \(\epsilon_t\) are i.i.d.	mbtsa::regression::diagnostics
What diagnostic visualizations are used to check linear regression assumptions?	(1) <b>Error Histogram:</b> check if residuals are normally distributed.<br>(2) <b>Q-Q Plot:</b> plot sorted residual quantiles vs. theoretical Normal quantiles — points should lie on a straight diagonal line if errors are Normal.<br>(3) <b>ACF Plot of Residuals:</b> check independence — significant spikes indicate the model failed to capture temporal dynamics in the data.	mbtsa::regression::diagnostics
What are the common sources of "bad residuals" (model misspecification)?	(1) <b>Wrong modality assumption:</b> errors may not be normally distributed.<br>(2) <b>Wrong relationship assumption:</b> the true relationship may be non-linear.<br>(3) <b>Missing external predictors:</b> important explanatory variables were omitted.<br>(4) <b>Missing temporal dynamics:</b> the model fails to account for dependencies between observations over time.	mbtsa::regression::diagnostics
What is basis expansion and why is the model still linear?	<b>Basis expansion</b> extends the linear model with non-linear transformations of predictors:<br>\(y_t = \beta_0 + \sum_{k=1}^K \beta_k \phi_k(x_t) + \epsilon_t\)<br><br>where \(\phi_k(\cdot)\) are chosen basis functions. The model is still <b>linear in the parameters</b> \(\beta_k\), so standard LSE/MLE solutions apply — only the design matrix changes to contain the transformed predictors.	mbtsa::regression::basis_expansion
What are common choices for basis functions in basis expansion?	(1) <b>Polynomials:</b> \(\phi_k(u_j) = u_{jt}^k\) (e.g., \(u_t, u_t^2, u_t^3\)).<br>(2) <b>Interaction terms:</b> \(\phi(u_i, u_j) = u_{it} \cdot u_{jt}\).<br>(3) <b>Radial Basis Functions:</b> depend on distance from a center point.<br>(4) <b>Fourier basis:</b> \(\sin(\omega t)\) and \(\cos(\omega t)\) for periodic patterns.	mbtsa::regression::basis_expansion
Write the multivariate linear regression model and its OLS solution.	<b>Model:</b> \(Y = XB + E\)<br><br>\(Y\): \(T \times q\) response matrix.<br>\(X\): \(T \times (p+1)\) design matrix.<br>\(B\): \((p+1) \times q\) parameter matrix (one column per response).<br>\(E\): \(T \times q\) error matrix.<br><br><b>OLS solution:</b> \(\hat{B} = (X^\top X)^{-1} X^\top Y\)<br><br>This is equivalent to performing \(q\) separate univariate regressions, one for each response variable.	mbtsa::regression::multivariate
What are the three components of a Generalized Linear Model (GLM)?	(1) <b>Random Component:</b> response \(y_t\) follows a distribution from the exponential family (Bernoulli, Poisson, Gamma, etc.).<br>(2) <b>Systematic Component:</b> linear predictor \(\eta_t = x_t^\top \beta\).<br>(3) <b>Link Function:</b> \(g(\mu_t) = \eta_t\), connecting the expected value \(\mu_t = \mathbb{E}[y_t]\) to the linear predictor. The inverse \(g^{-1}\) maps back: \(\mu_t = g^{-1}(\eta_t)\).	mbtsa::regression::glm
How is standard linear regression a special case of a GLM?	<b>Random component:</b> \(y_t \sim \mathcal{N}(\mu_t, \sigma^2)\).<br><b>Systematic component:</b> \(\eta_t = x_t^\top \beta\).<br><b>Link function:</b> the <b>identity link</b>, \(g(\mu_t) = \mu_t\).<br><br>Therefore \(\eta_t = \mu_t\), giving \(\mathbb{E}[y_t] = x_t^\top \beta\).	mbtsa::regression::glm
Describe the logistic regression model as a GLM.	<b>Response:</b> \(y_t \in \{0, 1\}\), \(y_t \sim \mathrm{Bernoulli}(\pi_t)\) where \(\pi_t = P(y_t = 1 \mid x_t)\).<br><b>Systematic component:</b> \(\eta_t = x_t^\top \beta\).<br><b>Link function (logit):</b> \(g(\pi_t) = \log\frac{\pi_t}{1 - \pi_t} = \eta_t\).<br><b>Inverse link (sigmoid):</b> \(\pi_t = \frac{1}{1 + \exp(-\eta_t)} = \sigma(\eta_t)\).	mbtsa::regression::logistic_regression
What is the gradient of the log-likelihood for logistic regression, and why is it solved iteratively?	<b>Log-likelihood:</b> \(\ell(\beta) = \sum_{t=1}^T \left[y_t(x_t^\top\beta) - \log(1 + \exp(x_t^\top\beta))\right]\)<br><br><b>Gradient:</b> \(\nabla_\beta \ell(\beta) = \sum_{t=1}^T (y_t - \pi_t)\, x_t\)<br><br>Setting this to zero does <b>not</b> yield a closed-form solution (unlike linear regression), so iterative methods like <b>gradient ascent</b> are used:<br>\(\beta_{\text{new}} = \beta_{\text{old}} + \alpha \sum_t (y_t - \pi_t)\, x_t\)	mbtsa::regression::logistic_regression
What is multimodal regression and what is its key assumption?	<b>Multimodal regression</b> models datasets where each observation has multiple types of data (modalities), e.g., binary \(y_{1t} \in \{0,1\}\) and continuous \(y_{2t} \in \mathbb{R}\). Each modality is modeled separately with its own parameters and distribution.<br><br><b>Key assumption — conditional independence:</b><br>\(p(y_{1t}, y_{2t} \mid x_t) = p(y_{1t} \mid x_t)\, p(y_{2t} \mid x_t)\)<br><br>This allows the total log-likelihood to decompose as the sum of per-modality log-likelihoods.	mbtsa::regression::multimodal
In multimodal regression, what happens when the conditional independence assumption fails?	If conditional independence between modalities is not reasonable, a common approach is to introduce a shared <b>latent variable</b> \(\epsilon_t\) that captures common dynamics or unobserved factors influencing all modalities. Conditional on both \(x_t\) and \(\epsilon_t\), the modalities are then assumed independent.	mbtsa::regression::multimodal
