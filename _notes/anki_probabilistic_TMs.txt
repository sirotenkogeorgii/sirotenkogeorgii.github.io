What is a Turing machine with auxiliary tape?	A deterministic TM with a special <b>read-only, one-way</b> tape (the auxiliary tape) containing an infinite binary sequence. The head can only move right. It is <b>total</b> if it terminates on all inputs and all auxiliary sequences. It is \(t(n)\)-time-bounded if total and runs \(\leq t(|w|)\) steps for almost all inputs \(w\) and all auxiliary sequences.	prob_TM::auxiliary_tape
How many bits can a \(t(n)\)-time-bounded TM with auxiliary tape read?	At most \(t(n)\) bits. A time-bounded machine can copy all bits it reads onto a work tape while respecting the time bound. The prefix of length \(t(n)\) of the auxiliary sequence is called the <b>auxiliary word</b>.	prob_TM::auxiliary_tape
How does the auxiliary tape model characterize NP?	\(L \in \text{NP}\) iff there exists a polynomially time-bounded TM \(M\) with auxiliary tape such that: \(w \in L \iff M\) accepts \(w\) for <b>some</b> auxiliary word \(z\). The auxiliary word plays the role of the NP certificate/witness.	prob_TM::auxiliary_tape
What is a probabilistic Turing machine (PTM)?	A <b>total</b> TM with auxiliary tape where the auxiliary tape is treated as a source of random bits (fair coin flips). The machine's behavior on input \(w\) depends on both \(w\) and the random word \(z\).	prob_TM::definition
Define the acceptance probability \(\text{accept}_M(w)\) of a PTM.	\[\text{accept}_M(w) = \frac{|\{z \in \{0,1\}^{t(n)} : M \text{ accepts on input } w \text{ and random word } z\}|}{2^{t(n)}}\] where \(t(n)\) is the number of random bits read. The rejection probability is \(\text{reject}_M(w) = 1 - \text{accept}_M(w)\).	prob_TM::definition
When does a PTM \(M\) accept an input \(w\)?	When \(\text{accept}_M(w) > \frac{1}{2}\), i.e., strictly more than half of all random words lead to acceptance. The language recognized by \(M\) is \(L(M) = \{w : \text{accept}_M(w) > \frac{1}{2}\}\).	prob_TM::definition
Define the error probability of a PTM \(M\) on input \(w\) w.r.t. language \(L\).	\[\text{error}_M^L(w) = \begin{cases} \text{reject}_M(w) & \text{if } w \in L \\ \text{accept}_M(w) & \text{if } w \notin L \end{cases}\] It measures the probability that \(M\) gives the wrong answer.	prob_TM::definition
Give the alternative definition of PTMs via nondeterministic computation trees.	A PTM can be defined as an NTM where each inner node has exactly 2 children and all leaves on a given input have the same depth. Acceptance probability = (number of accepting leaves) / (total leaves). This is equivalent to the auxiliary tape model.	prob_TM::definition
Define PP (Probabilistic Polynomial Time).	\(\text{PP} = \{L : L = L(M) \text{ for some polynomially time-bounded PTM } M\}\). A PTM accepts \(w\) if \(\text{accept}_M(w) > \frac{1}{2}\). The acceptance probability can be arbitrarily close to \(\frac{1}{2}\).	prob_TM::PP_BPP
Define BPP (Bounded-error Probabilistic Polynomial Time).	\(\text{BPP} = \{L : L = L(M) \text{ for some poly-time-bounded PTM } M \text{ with error probability} \leq \frac{1}{3}\}\). Requires a constant gap from \(\frac{1}{2}\) — the machine must be "reliably" correct.	prob_TM::PP_BPP
What is the key difference between PP and BPP?	<b>PP</b> allows acceptance probability arbitrarily close to \(\frac{1}{2}\) (e.g., \(\frac{1}{2} + 2^{-n}\)). <b>BPP</b> requires a constant gap: error \(\leq \frac{1}{3}\), so acceptance probability is \(\geq \frac{2}{3}\) for members and \(\leq \frac{1}{3}\) for non-members.	prob_TM::PP_BPP
State and prove: \(\text{BPP} \subseteq \text{PP} \subseteq \text{PSPACE}\).	\(\text{BPP} \subseteq \text{PP}\): by definition (error \(\leq \frac{1}{3} < \frac{1}{2}\)).<br><br>\(\text{PP} \subseteq \text{PSPACE}\): a deterministic poly-space TM simulates the PTM on <b>each</b> random word \(z\) one-by-one, reusing space. Uses a binary counter (\(t(n)\) bits) to count acceptances. Accepts iff count \(> 2^{t(n)-1}\). Total space: \(O(t(n))\) = polynomial.	prob_TM::PP_BPP
Prove: \(\text{NP} \subseteq \text{PP}\).	Given \(L \in \text{NP}\) with poly-time verifier \(M\), construct PTM \(P\): take random word \(r = bz\) (one extra bit). If \(b=1\), always accept. If \(b=0\), simulate \(M\) on \(w\) with auxiliary word \(z\), accept iff \(M\) accepts.<br><br>If \(w \notin L\): accept prob = exactly \(\frac{1}{2}\) (only from \(b=1\) case).<br>If \(w \in L\): accept prob \(> \frac{1}{2}\) (from \(b=1\) plus some \(b=0\) cases).<br>So \(L(P) = L\) and \(P\) is poly-time.	prob_TM::PP_BPP
State: PP and BPP are closed downwards under p-m-reducibility.	If \(A \leq_p B\) via \(g\) and \(B \in \text{PP}\) (or BPP), then \(A \in \text{PP}\) (or BPP). Proof: PTM \(M_A\) on input \(w\) computes \(g(w)\) then simulates the PTM for \(B\) on \(g(w)\). Time: \(p_g(n) + p_M(p_g(n))\) = polynomial.	prob_TM::PP_BPP
State and prove the lemma: every PP language has error strictly \(< \frac{1}{2}\).	Given PTM \(M\) for \(L \in \text{PP}\) reading \(p(n)\) random bits, construct \(M'\) reading \(2p(n)\) bits split into Block A and Block B. \(M'\) accepts iff \(M\) accepts (using Block A) <b>AND</b> Block B \(\neq 0^{p(n)}\).<br><br>If \(w \notin L\): accept prob \(\leq \frac{1}{2}(1 - 2^{-p(n)}) < \frac{1}{2}\).<br>If \(w \in L\): accept count \(= (2^{p(n)-1}+k)(2^{p(n)}-1) > 2^{2p(n)-1}\), so prob \(> \frac{1}{2}\).	prob_TM::PP_properties
Prove: PP is closed under complement.	By the lemma, any \(L \in \text{PP}\) has a PTM with error \(< \frac{1}{2}\). This means accept prob \(> \frac{1}{2}\) for members and \(< \frac{1}{2}\) for non-members (no ties). Swapping accept/reject gives a PTM recognizing \(\bar{L}\) with accept prob \(> \frac{1}{2}\) for non-members of \(L\).	prob_TM::PP_properties
Prove: PP is closed under symmetric difference.	Let \(M, M'\) recognize \(L, L'\) with errors \(\frac{1}{2} - \epsilon_w\) and \(\frac{1}{2} - \epsilon'_w\) (\(\epsilon, \epsilon' > 0\)). Machine \(\tilde{M}\) simulates both with independent random bits, accepts iff exactly one accepts (XOR).<br><br>\(\tilde{M}\) is correct iff both simulations are correct or both wrong. Probability: \((\frac{1}{2}+\epsilon)(\frac{1}{2}+\epsilon') + (\frac{1}{2}-\epsilon)(\frac{1}{2}-\epsilon') = \frac{1}{2} + 2\epsilon\epsilon' > \frac{1}{2}\).	prob_TM::PP_properties
What is MAJ (majority satisfiability problem)?	\(\text{MAJ} = \{\phi : \phi \text{ is a CNF formula satisfied by strictly more than half of all assignments}\}\)	prob_TM::PP_properties
State: MAJ is PP-complete. Sketch the proof.	<b>MAJ \(\in\) PP</b>: PTM interprets random bits as a variable assignment, accepts iff it satisfies \(\phi\). Accept prob \(> \frac{1}{2}\) iff majority of assignments satisfy \(\phi\).<br><br><b>MAJ is PP-hard</b>: Like Cook's theorem — given a PTM \(M\) for \(L \in \text{PP}\), construct CNF \(\phi_w\) with a bijection between satisfying assignments and accepting random words. Then \(w \in L \iff \phi_w \in \text{MAJ}\).	prob_TM::PP_properties
Is it known whether PP is closed under union or intersection?	No, this is an open question. PP is known to be closed under complement and symmetric difference.	prob_TM::PP_properties
Is it known whether BPP has complete languages?	No, it is not known whether complete languages for BPP exist.	prob_TM::PP_properties
State the Probability Amplification theorem for BPP.	The following are equivalent:<br>(i) \(L \in \text{BPP}\).<br>(ii) \(L\) is recognized by a poly-time PTM with error \(\leq \frac{1}{2} - \epsilon\) for some \(\epsilon > 0\).<br>(iii) For <b>every</b> polynomial \(p\), there is a poly-time PTM with error \(\leq 2^{-p(n)}\) recognizing \(L\).<br><br>Error can be made exponentially small while remaining in polynomial time.	prob_TM::BPP_properties
Explain the proof technique for probability amplification.	Run \(m = 2t(n)+1\) independent simulations of the original PTM and take a <b>majority vote</b>. Error occurs iff \(\leq t(n)\) simulations are correct. By Chernoff-type bound, error probability \(\leq (1 - 4\epsilon^2)^{t(n)}\). Choose \(t(n) = c \cdot p(n)\) for a suitable constant \(c\) to get error \(\leq 2^{-p(n)}\).	prob_TM::BPP_properties
State the closure properties of BPP.	BPP is closed under:<br>- <b>Complement</b> (swap accept/reject)<br>- <b>Union</b><br>- <b>Intersection</b><br>- <b>Symmetric difference</b><br>- All binary set-theoretic operations<br><br>Key technique: amplify error to \(\leq \frac{1}{6}\) for each machine, then union of errors \(\leq \frac{1}{3}\).	prob_TM::BPP_properties
Prove BPP is closed under intersection.	Let \(L, L' \in \text{BPP}\). By amplification, get PTMs \(M, M'\) with error \(\leq \frac{1}{6}\) each. Build PTM that simulates both (independent random bits) and accepts iff both accept. Error \(\leq \frac{1}{6} + \frac{1}{6} = \frac{1}{3}\) by union bound. So \(L \cap L' \in \text{BPP}\).	prob_TM::BPP_properties
What is an advice function? Define P/poly.	An <b>advice function</b> maps natural numbers to binary words. A TM \(M\) with auxiliary tape recognizes \(L\) with advice \(a(n)\) if for every input \(w\) of length \(n\): \(M\) accepts \(w\) on auxiliary word \(a(n)\) iff \(w \in L\).<br><br>\(\text{P/poly}\) = languages recognizable by a poly-time TM with auxiliary tape with some advice function. The advice can always be chosen of polynomial length.	prob_TM::P_poly
Prove: \(\text{BPP} \subseteq \text{P/poly}\).	Let \(L \in \text{BPP}\) with PTM \(M\) having error \(\leq 2^{-2n}\). For each input of length \(n\), at most a \(2^{-2n}\) fraction of random words are bad. Over all \(2^n\) inputs, at most \(2^n \cdot 2^{-2n} = 2^{-n}\) fraction is bad for <b>some</b> input. So a <b>good</b> random word exists that works for all inputs of length \(n\). Use it as the advice \(a(n)\).	prob_TM::P_poly
Define the polynomial hierarchy: \(\Sigma_k^p\) and \(\Pi_k^p\).	\(\Sigma_k^p\): languages \(L\) where \(w \in L \iff \exists_{y_1}^{p(n)} \forall_{y_2}^{p(n)} \exists_{y_3}^{p(n)} \cdots Q_k y_k\, ((w, y_1, \ldots, y_k) \in B)\) for some \(B \in \text{P}\) and polynomial \(p\). Quantifiers alternate starting with \(\exists\).<br><br>\(\Pi_k^p\): same but starts with \(\forall\).	prob_TM::PH
What are the base cases and basic properties of the polynomial hierarchy?	\(\Sigma_0^p = \Pi_0^p = \text{P}\), \(\Sigma_1^p = \text{NP}\), \(\Pi_1^p = \text{co-NP}\).<br><br>\(L \in \Sigma_k^p \iff \bar{L} \in \Pi_k^p\).<br>\(\Sigma_k^p \cup \Pi_k^p \subseteq \Sigma_{k+1}^p \cap \Pi_{k+1}^p\).<br>\(\text{PH} = \bigcup_k \Sigma_k^p \subseteq \text{PSPACE}\).	prob_TM::PH
What is a collapse of the polynomial hierarchy?	If \(\Sigma_k^p = \Sigma_{k+1}^p\), then \(\Sigma_k^p = \text{PH}\) — the hierarchy "collapses" to level \(k\). If \(\text{PH} = \text{PSPACE}\), then PH collapses (since PSPACE has a complete problem and each \(\Sigma_k^p\) is closed under p-m-reductions).	prob_TM::PH
State: \(\text{BPP} \subseteq \Sigma_2^p \cap \Pi_2^p\).	\(\text{BPP} \subseteq \Sigma_2^p\): for \(L \in \text{BPP}\), there exist \(B \in \text{P}\) and polynomial \(p\) such that \(w \in L \iff \exists_x^{p(n)} \forall_y^{p(n)} ((w,x,y) \in B)\).<br><br>Since BPP is closed under complement, \(\bar{L} \in \text{BPP} \subseteq \Sigma_2^p\), so \(L \in \Pi_2^p\). Hence \(\text{BPP} \subseteq \Sigma_2^p \cap \Pi_2^p\).	prob_TM::PH
Sketch the proof of \(\text{BPP} \subseteq \Sigma_2^p\) using XOR shifts.	Amplify error to \(2^{-n}\). Let \(U\) = set of random words making \(M\) accept \(w\). Statement (4.3): \(\exists v_1 \cdots v_{p(n)} \forall z\, (z \in U \oplus v_1 \lor \cdots \lor z \in U \oplus v_{p(n)})\).<br><br><b>If \(w \notin L\)</b>: \(|U| \leq 2^{p(n)-n}\), so \(p(n)\) shifts can't cover all \(2^{p(n)}\) words → (4.3) is false.<br><b>If \(w \in L\)</b>: \(|U| \geq (1-2^{-n})2^{p(n)}\). Probabilistic argument shows good shifts exist → (4.3) is true.	prob_TM::PH
Summarize the chain of inclusions involving probabilistic classes.	\(\text{P} \subseteq \text{BPP} \subseteq \Sigma_2^p \cap \Pi_2^p \subseteq \text{PH} \subseteq \text{PSPACE}\)<br>\(\text{NP} \subseteq \text{PP} \subseteq \text{PSPACE}\)<br>\(\text{BPP} \subseteq \text{PP}\)<br>\(\text{BPP} \subseteq \text{P/poly}\)	prob_TM::summary
